# KAPCO PayPeople Offline APK

This repository contains an offline Android WebView demo app for KAPCO HRMS / PayPeople-style testing.

## Login

Email: `admin@kapco.com.pk`  
Password: `K@pco123`

## Included modules

- Dashboard
- Employees
- Attendance
- Leave management
- Payroll / salary slip
- Reports

## APK build

Open the Actions tab and run **Build KAPCO PayPeople APK**. After completion, download the artifact named **KAPCO-PayPeople-APK** and install `app-debug.apk` on Android.
